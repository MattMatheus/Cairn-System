# Knowledge Workflow

This page explains the knowledge plane. Use `workspace/docs/HUMANS.md` for the main workspace navigation path.

## Claim Lifecycle
`idea -> candidate -> active-test -> supported|falsified|parked`

## Promotion Conditions
- `candidate -> active-test`: test plan exists.
- `active-test -> supported|falsified`: one or more artifact links exist.

## Canonical Zones
- Concepts: `workspace/research/`
- Claims: `workspace/research/`
- Artifacts: `workspace/research/`
- Decisions: `workspace/research/`
- Syntheses: `workspace/research/`

## Agent Routing
- New intake: `workspace/agents/queue/`
- Proposed canonical promotions: `workspace/agents/staging/`
- Human-approved promotions only.
