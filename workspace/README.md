# Workspace

Canonical markdown workspace surface for Cairn.

This is the practical daily-use environment for developers using AthenaWork with AthenaMind-backed assistance.

## Start Here

1. `workspace/docs/HUMANS.md`
2. `workspace/docs/README.md`
3. `workspace/docs/EXAMPLE-WORKFLOW.md`

## Main Areas

- `workspace/docs/` for human workflow and policy
- `workspace/agents/` for queue, control, and agent-facing material
- `workspace/templates/` for task, claim, artifact, and research templates
- `workspace/work/` for work notes and examples
- `workspace/research/` for claims, artifacts, and example evidence

## Runtime Boundary

Committed workspace guidance lives here.

Repo-local runtime state belongs under `.athena/`, especially:

- `.athena/workspace/`
- `.athena/memory/`
- `.athena/runs/`
- `.athena/artifacts/`
