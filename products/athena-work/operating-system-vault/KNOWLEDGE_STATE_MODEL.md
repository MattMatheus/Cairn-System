# Knowledge State Model

## Claim Lifecycle
`idea -> candidate -> active-test -> supported|falsified|parked`

## Required Conditions
- `candidate -> active-test`: test plan present.
- `active-test -> supported|falsified`: linked artifact evidence present.

## Canonical Knowledge Paths
- Concepts: `80 Research/10 Concepts/`
- Claims: `80 Research/20 Claims/`
- Artifacts: `80 Research/30 Artifacts/`
- Decisions: `80 Research/40 Decisions/`
- Syntheses: `80 Research/50 Syntheses/`

## Agent Routing
- Raw agent intake: `70 Agents/00 Queue/`
- Proposed promotions: `70 Agents/90 Staging/`
- Human-approved promotion to canonical research zones.
